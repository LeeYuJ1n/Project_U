# INFORMAION
아래와 같이 파일 유형 이름을 통일시켜 주시기 바랍니다</br>
추가사항이 있을 경우 업데이트 하겠습니다</br>

○ File Type Name</br>
Material → MT_</br>
BluePrint → BP_</br>
Animation_BluePrint → ABP_</br>
Montage → MTG_</br>
NiagaraSystem → NGS_</br>
입력 매칭 컨텍스트 → IMC_</br>
입력 액션 → IC_</br>
블랙보드 → BB_</br>
비헤이비어 트리 → BT_</br>
위젯 블루프린트 → WBP_
